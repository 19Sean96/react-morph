# Getting Started

## Intro

Import the `useMorph` hook.

```js
import { useMorph } from 'react-morph';
```

Call the hook in you function component.

```js
const morph = useMorph(options);
```

Then spread the props to the elements you want to morph.

```jsx
<img {...morph} src="larva.png" width="50">
```

```jsx
<img {...morph} src="butterfly.png" width="80">
```

You should handle the state as you normally would.

Make sure you have just **one element redered at same time**.

## Demo

<Playground>
  {() => {
	  // Handle toggle state as you normally would.
	  const [toggle, setToggle] = useState(true)
	  const morph = useMorph();

      return (
        <div>
          <button onClick={() => setToggle(!toggle)}>Let's morph!</button>
					<br />
          {toggle && <img key="one" {...morph} src={larva} width="30" />}
  				<br />
  				<br />
  				<br />
          {!toggle && <img key="two" {...morph} src={ butterfly} width="80" />}
        </div>
      )
    }

}
